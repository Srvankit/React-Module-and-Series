📁 Important Files Explained

App.jsx
Main React component where most logic starts.

main.jsx
Entry point of the application. This is where React renders the <App /> component into the DOM.

index.css
Global CSS file. Used for importing Tailwind CSS and writing global styles.

vite.config.js
Configuration file when using Vite. Used to configure plugins like Tailwind CSS.

package.json
Contains:

Project dependencies

Scripts (like npm run dev)

Project metadata

📦 How to Run Any Project Locally

Navigate into the project folder:

cd project-folder-name

Install dependencies:

npm install

Start development server:

For Vite projects:
npm run dev

For Create React App projects:
npm start
